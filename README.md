Proyek ini berisi hasil pengerjaan technical test untuk posisi Data Analyst di **Agate**. Fokus utama dari analisis ini adalah melakukan eksplorasi data, mengidentifikasi tren utama, serta memberikan rekomendasi berbasis data (data-driven insights) untuk mendukung pengambilan keputusan bisnis.

## 📌 Gambaran Proyek

Dataset yang dianalisis mencakup data transaksi/user behavior yang memerlukan pemrosesan mendalam untuk menjawab tantangan bisnis tertentu. Analisis ini mencakup:
- **Data Cleaning & Preprocessing:** Menangani nilai yang hilang, inkonsistensi tipe data, dan pembersihan *outliers*.
- **Exploratory Data Analysis (EDA):** Visualisasi distribusi data, korelasi antar variabel, dan identifikasi pola perilaku.
- **Business Insights:** Penarikan kesimpulan strategis berdasarkan temuan statistik.

## 🛠️ Teknologi & Library

Proyek ini dikembangkan menggunakan workflow berbasis terminal dan environment Python:
- **Bahasa:** Python 3.x
- **Analisis Data:** Pandas, NumPy
- **Visualisasi:** Matplotlib, Seaborn
- **Environment:** Jupyter Notebook / Google Colab
- **Version Control:** Git
